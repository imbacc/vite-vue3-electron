# vite-vue3-electron

##### 为了可以单独打包做的分歧打包
##### 注意: 渲染端如果作为web服务的话需要写兼容代码 防止报electron内部错误
##### pnpm run v-dev 或 pnpm run e-dev
##### 打包成应用pnpm run build

vite <https://vitejs.dev>
<br/>
vue3 <https://v3.cn.vuejs.org>
<br/>
electron  <https://cloud.tencent.com/developer/doc/1070> <https://www.electronjs.org>
<br/>
### vite和electron都启动
`pnpm run dev`
<br/>
### 启动vite vue3
`pnpm run v-dev`
<br/>
### 启动electron webview...
`pnpm run e-dev`
