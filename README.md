# vite-vue3-electron

##### 为了单独打包,web端和桌面应用
##### 打包成应用pnpm run build

vite <https://vitejs.dev>
<br/>
vue3 <https://v3.cn.vuejs.org>
<br/>
electron  <https://cloud.tencent.com/developer/doc/1070> <https://www.electronjs.org>
<br/>
### web和electron都启动
`pnpm run dev`
### 只启动web
`pnpm run v-dev`
<br/>
### 只编译web
`pnpm run v-build`
<br/>
### 编译桌面应用
`pnpm run build`
